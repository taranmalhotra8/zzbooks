# Social Media Asset Fonts

Place the following font files in this directory for Satori rendering:

- Inter-Regular.ttf
- Inter-Bold.ttf
- JetBrainsMono-Regular.ttf

Download from:
- Inter: https://github.com/rsms/inter/releases
- JetBrains Mono: https://github.com/JetBrains/JetBrainsMono/releases

These fonts are required by the social media asset generator (_social/generate.ts).
